withEase

  • Inherited from Seekable.withEase

    • Defined in index.ts:185

    Returns a new Seekable with the given easing function applied.

    Note that if you attempt to combine eases, e.g. you call withEase on a Tween which already has an ease, the performance of the code will start to get really bad. Consider instead creating a new ease function that does what you want, and setting the Tween's ease to your new function instead.

    Parameters

    • ease: EasingFunction

      The easing function to use. The easing function can be any function with the signature (x) => number where x is a number in the range 0 <= x <= 1
