Class ObjectPropertyAnimationTarget<T, O, P>

Defines an animation target for a completely separate object's property.

WARNING: This is an ugly hack which is provided for compatibility purposes; ideally you should ABSOLUTELY design your own objects which make use of the proper AnimationTarget and not use ObjectPropertyAnimator.

Be careful when using objects of this type, because behavior may be odd.
